# 손박사 🔬 — 포켓몬GO 디스코드 봇

포켓몬GO 관련 정보(CP 계산, 실시간 레이드, 대기열, 내전 팀 관리)를 슬래시 명령어로 바로 확인할 수 있는 디스코드 봇입니다.

## 기능

### `/포켓몬 이름:이상해씨`
한글 포켓몬 이름을 입력하면 **레벨 20 / 25, IV 100%(15/15/15) 기준 CP**를 알려줍니다. 레이드·부화·필드리서치는 레벨이 20 또는 25로 고정되고 최소 IV가 10/10/10(66%)이라, 잡은 포켓몬이 "헌드로"인지 CP만 보고 가늠할 수 있습니다. 오타가 있어도 가장 비슷한 이름을 찾아줍니다.

### `/레이드`
**현재 진행 중인 레이드 보스**를 실시간으로 조회합니다. 1성/3성/5성/메가로 등급별 정리되고, 각 보스마다 IV 100% 기준 Lv20/Lv25 CP와 이로치 가능 여부(✨)가 함께 표시됩니다. 지역폼·섀도우·메가 접두어가 붙은 이름은 자동으로 한글화됩니다 (코스튬 이벤트 한정 이름은 영문 그대로 표시될 수 있습니다).

### `/줄서기 인원:1385`
현재 대기 인원수를 입력하면 예상 대기 시간을 알려줍니다. (분당 약 23명 처리 기준)

### `/내전`
1대1~5대5 중 형식을 고르고 플레이어 이름을 입력하면 랜덤으로 두 팀을 나눠줍니다. 승리 팀 버튼을 누르면 소속 전원의 승수가 기록되고, **개인별 통산 전적 순위**가 임베드에 함께 표시됩니다. 팀을 계속 다시 섞어도 전적은 유지됩니다.
- 🔴 A팀 승리 / 🔵 B팀 승리 — 승수 기록
- 🔁 다시 섞기 — 같은 명단으로 재섞기 (전적 유지)
- ✏️ 명단 수정 — 이름 변경 (겹치는 이름은 기존 전적 승계)
- ♻️ 리셋 — 전적 포함 전체 초기화

### `/운세 생년월일:20000101 태어난시간:1430`
생년월일(+태어난시간)을 입력하면 오늘 하루 포켓몬GO 테마의 재미용 운세를 봅니다 (종합운/이로치 조우운/레이드운/알까기운/친밀도운). 같은 사람은 하루 동안 같은 결과가 나오고, 다음 날 다시 물어보면 바뀝니다. 실제 사주와는 무관한 엔터테인먼트 콘텐츠입니다.

## 데이터 출처 (Credits)

- **포켓몬 종족값**: [PoGoAPI](https://pogoapi.net)
- **한글 포켓몬 이름**: [PokeAPI](https://pokeapi.co) (공식 다국어 지원)
- **실시간 레이드 보스 정보**: [ScrapedDuck](https://github.com/bigfoott/ScrapedDuck) — [LeekDuck.com](https://leekduck.com)을 주기적으로 스크래핑해 공개하는 프로젝트로, 원본 데이터는 LeekDuck.com에서 제공합니다. ScrapedDuck 이용 조건에 따라 이 크레딧을 명시합니다.

## 참고 / 한계

- 메가진화·알로라·가라르 등 특수 폼은 `/포켓몬` CP 계산에서 기본 폼 종족값만 지원합니다.
- `/레이드`의 한글 번역은 "지역폼/섀도우/메가 + 종 이름" 패턴만 지원하며, 코스튬 이벤트 전용 이름(예: 기념일 한정 피카츄)은 매칭 데이터가 없어 영문 그대로 표시됩니다.
- `/운세`는 실제 사주·운세와 무관한 재미용 콘텐츠입니다.
