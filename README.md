# 손박사 🔬 사용 설명서

포켓몬GO에서 레이드·부화·필드리서치로 포켓몬을 잡았을 때, **"이거 헌드로(IV 100%) 아니야?"** 싶을 때 CP만 보고 바로 확인할 수 있게 도와주는 디스코드 봇입니다.

## 사용법

디스코드 채팅창에 아래처럼 입력하세요.

```
/포켓몬 이름:이상해씨
```

- `이름`에는 확인하고 싶은 포켓몬의 **한글 이름**을 입력하면 됩니다.
- 슬래시(`/`)와 함께 입력하면 자동완성 목록이 뜨는데, 거기서 선택해도 됩니다.
- 오타가 조금 있어도 손박사가 가장 비슷한 이름을 찾아서 알려줍니다.

## 응답 예시

`/포켓몬 이름:이상해씨` 입력 시:

> **이상해씨 (Bulbasaur) CP 정보**
> 레이드 / 부화 / 필드리서치 포획 시 (레벨 20 또는 25, 최소 IV 10/10/10 보장) 예상 CP 범위
>
> 🌤 **레벨 20 (날씨 부스트 없음)**
> IV100%(15/15/15): **557 CP**
> IV66%(10/10/10, 최소): 484 CP
>
> ☀️ **레벨 25 (날씨 부스트)**
> IV100%(15/15/15): **695 CP**
> IV66%(10/10/10, 최소): 605 CP

## 이 정보를 어떻게 활용하나요?

레이드·부화·필드리서치로 잡은 포켓몬은 **레벨이 20 또는 25로 고정**되고, **최소 IV가 10/10/10(66%)로 보장**됩니다. (날씨 부스트가 걸려 있었으면 레벨 25, 아니면 레벨 20)

1. 포켓몬을 잡은 뒤 화면에 뜨는 **CP**를 확인합니다.
2. 손박사에게 `/포켓몬` 명령어로 그 포켓몬을 조회합니다.
3. 잡은 CP가 손박사가 알려준 **레벨20(또는25) IV100% CP**와 같거나 매우 가까우면 헌드로(100%)일 가능성이 높습니다.
4. 애매하면(범위 사이 값이면) 게임 내 포켓몬 상세화면에서 **팀 리더 평가**(별 3개, "경이로운" 등)를 같이 참고하거나, IV 계산기 앱(Calcy IV 등)에 CP+HP를 같이 넣어 정확히 좁혀보세요. CP 하나만으로는 100% 확정이 어려운 경우도 있습니다.

## 참고 사항

- 날씨 부스트 여부는 레이드 배틀 화면에 표시되는 날씨 아이콘으로 확인할 수 있어요.
- 현재는 메가진화·알로라·가라르 등 특수 폼은 지원하지 않고, 기본 폼 포켓몬만 조회됩니다.
- 봇은 여러분이 입력한 포켓몬 이름 외에 별도의 개인정보를 저장하지 않습니다.
